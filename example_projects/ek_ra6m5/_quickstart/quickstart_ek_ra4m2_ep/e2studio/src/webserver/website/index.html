Introduction

This embedded web server is running on the RA6M5 MCU and is implemented using Renesas Flexible Software Package, FreeRTOS+TCP, and open source Webio.

This web server demonstrates how users can easily create a professional, user-friendly, and platform-independent user interface for products/applications that require remote control and monitoring.


Instructions
  1. Press the physical user buttons S1 and/or S2 on the EK-RA6M5 board. The blue user LED's intensity and/or frequency will change. The intensity and/or frequency will refresh in the Display Console. This demostrates the communication from the server to the client.
  2. Press the software user buttons S1 and/or S2 on the web server Control Panel. The blue user LED's intensity and/or frequency will change. The intensity and/or frequency will refresh in the Display Console. This demostrates the communication from the client to the server.

Please refer to the EK-RA6M5 Quick Start Guide and Quick Start Example Project source code for more details.
